# BC STR Comply

Free compliance tracker for short-term rental hosts in British Columbia.

## Quick Start

```bash
npm install
npm run dev
```

## Deploy

```bash
# Push to GitHub, Vercel auto-deploys
git add .
git commit -m "update"
git push
```

## Features

- Interactive compliance checklist (Vancouver, Victoria, Kelowna, West Vancouver)
- Risk scoring with real-time updates
- Deadline tracker (municipal + provincial)
- Fines & fees breakdown
- Regulation updates feed
- AI chatbot for compliance questions
- Email capture for deadline reminders
- Affiliate/partner infrastructure
- 4 languages (English, Chinese, Punjabi, French)
